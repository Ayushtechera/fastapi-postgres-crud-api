# 🚀 FastAPI PostgreSQL CRUD API

A simple learning project built to understand FastAPI, PostgreSQL, SQLAlchemy ORM, and REST API development.

![Project Screenshot](assets\project-preview.png)

## ✨ Features

- Create Products
- View All Products
- Get Product by ID
- Update Products
- Delete Products
- PostgreSQL Database Integration
- RESTful API Endpoints

## 🛠️ Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic
- Uvicorn
- React

## ⚙️ Installation

```bash
git clone https://github.com/Ayushtechera/fastapi-postgres-crud-api.git
cd fastapi-postgres-crud-api
```

## ▶️ Run the Project

### Backend

```bash
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm start
```
